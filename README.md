### Identificación del escenario práctico en el que voy a establecer la práctica:

La idea principal es crear una aplicación simple del juego del STOP. Digo simple porque esta no va a constar con puntuajes
ni mucho menos con revision de las palabras. Se verificará que las palabras empiecen por la letra especificada y poco más.
Se sumará un punto por palabra escrita, no se va a evaluar si las palabras son repetidas.

La letra será elegida de manera aleatoria por el servidor.

Las salas serán de hasta 5 jugadores.

El usuario solo necesitará introducir un nombre representativo.
